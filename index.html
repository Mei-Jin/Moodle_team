Welcome to Moodle magpie team home page.

We are a group of students taking a Master in Computer Science at University College Dublin, in Ireland.

As part of the course, we are doing a development project and have chosen to explore and build some MyMoodle enhancements with a view to primarily focusing on aggregation blocks but looking for feedback and ideas for any other blocks.

We have placed the initial ideas from our project sponsor in the following Google Document http://tinyurl.com/d4l5ntn. Once we have done some more research and got more feedback the goal is to prioritise them and then build the ones we can within the duration of the project.

Look for Feedback

A quick summary of the ideas (which are site-wide blocks just for MyMoodle) are:

  1. Student My Assignments Block
  2. My Timeline Block
  3. Recent Communication Block
  4. MY Grades Block
  5. Site Activity Block
  6. Teacher Tasks Block
  7. Common Admin Tasks block (for descriptions see the document)

We would love to get any feedback on the existing ideas, what you think of them and if you have any suggestions regarding them or any other blocks you would really like to see developed.
